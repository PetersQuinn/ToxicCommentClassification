# Tabular Exports

This folder holds the tabular feature branch for the larger toxic comment project.

We keep a few small tracked artifacts here:
- `metadata/dense_feature_dictionary.csv`
- `metadata/run_config.json`
- `metadata/run_metadata.json`
- `metadata/tfidf_metadata.json`
- `diagnostics/identity_group_counts_train.csv`

Dense parquet tables, TF-IDF matrices, vectorizer files, and row-alignment exports are generated locally and ignored.
