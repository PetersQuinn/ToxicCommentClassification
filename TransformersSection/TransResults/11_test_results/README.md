# 11_test_results

I use this folder for the clean test-set comparison between the shared multilabel model and the six one-label binary models.

What I would open first
- `overall_test_results_summary.json`
- `multilabel_vs_binary_test_comparison.csv`
- `ap_by_label_multilabel_vs_binary.png`
- `ap_delta_binary_minus_multilabel.png`

This is the best place to see where the shared transformer held up and where a label-specific binary model pulled ahead.
