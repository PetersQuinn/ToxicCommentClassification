# 08_experiment_summary

I use this folder for the first cross-run summary of the transformer section.

What is here
- `model_comparison_summary.csv` and `model_comparison_summary.json` give me one table of the main validation and test metrics plus the main cost stats for each transformer variant.
- `primary_test_metric_by_variant.png` is the quickest side-by-side performance plot.

This is where I compare the shared multilabel model against the six one-label binary runs. I still treat this as one section of the larger project, not the full project summary.
